# Exercise 4.5: Schema Design Patterns

## Setup

```bash
cd claude-architect-student-exercises/exercises/4.5-schema-design/starter/
npm install
cp .env.example .env
# Edit .env and add your API key
```

## Running

```bash
node extractor.js basic       # Test with the basic schema
node extractor.js improved    # Test with your improved schema
```

## Files

- `schema-improved.js` — YOUR file. Design the improved schema here.
- `schema-basic.js` — Basic schema for comparison. Don't modify.
- `sample-documents.js` — 5 test documents. Don't modify.
- `extractor.js` — Extraction runner with quality checks. Don't modify.
